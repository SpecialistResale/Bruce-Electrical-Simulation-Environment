# Safety Rules

Bruce Electrical Simulator is a sandboxed analysis environment.

## Allowed use
- Engineering education
- Conceptual field analysis
- Formula exploration
- Visualization of simplified electrical models

## Prohibited use
- Live power system switching procedures
- Real hardware control or actuation
- Operational sabotage or attack guidance
- Weaponized electromagnetic design workflows

## Response policy
If a request moves toward unsafe real-world operations, the system must decline and provide safe conceptual alternatives.
